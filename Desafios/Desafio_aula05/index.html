CALCULADORA

Para utilizar a calculadora basta abrir o console e efetuar os seguintes comandos:

- CALCULADORA.multiplicacao(AQUI OS VALORES QUE DESEJA, USANDO VIRGULA PARA SEPARA-LOS);
-CALCULADORA.subtracao(AQUI OS VALORES QUE DESEJA, USANDO VIRGULA PARA SEPARA-LOS);
-CALCULADORA.divisao(AQUI OS VALORES QUE DESEJA, USANDO VIRGULA PARA SEPARA-LOS);
-CALCULADORA.exponenciacao(AQUI OS VALORES QUE DESEJA, USANDO VIRGULA PARA SEPARA-LOS);
-CALCULADORA.historico --> PARA SABER O HISTÓRICO DOS RESULTADOS
-CALCULADORA.clear ---> PARA LIMPAR SEU HISTÓRICO
